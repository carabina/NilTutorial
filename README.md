## Requirements
  Swift > 3.0
## Installation

NilTutorial is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'NilTutorial'
```
## Usages
```swift
import NilTutorial

            // Add skip button action here
```
